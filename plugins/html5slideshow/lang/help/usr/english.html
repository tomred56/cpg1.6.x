Display slideshow in a new (pop-up) window

When starting a slideshow, it will be displayed in a new window/tab. When the slideshow is stopped, the window/tab will be removed.


Image size

Self explanatory - the image size used for the slideshow.


Image transition

Select the image transition type:
   none - each image is just immediately replaced by the next
   dissolve - current image fades out and new image fades in
   slide - current image slides offscreen to the left while new image slides in from the right.


Shuffle image order

Each time the slideshow is run, the order of images displayed would be different.


Start show in "play" mode

When the slideshow is started, images will automatically begin advancing. Otherwise, a keyboard, mouse or touch action would be needed to advance to the next image.
