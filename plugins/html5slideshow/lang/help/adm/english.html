Display new slideshow icon at album header

Selecting this will cause a slideshow icon () to be displayed next to the album name when viewing the album. Clicking on the icon will begin a slideshow of the album's content (currently, images only).


Modify standard slideshow button action

This option will cause the action for the standard CPG slideshow icon () to run a HTML5 Slideshow rather than the inbuilt CPG slideshow.


Users can configure their slideshow settings

Selecting this will allow users to configure their own slideshow viewing preferences, both for each of their albums and globally for all albums they view.
Order of precedence for slideshow parameters is:
   CPG Site Settings <-- User Default Settings <-- Album Specific Settings
